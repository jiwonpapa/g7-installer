G7 Installer

G7 설치 마법사

서버 점검부터 도메인 연결 검증 리포트까지 단계별로 진행합니다.

연결 시작 중
현재 버전은 apt 패키지 설치, Nginx 도메인 연결, PHP-FPM/DB 튜닝, DB 계정 생성, Nginx 기준 SSL 발급/갱신 검증, 설정 안내서 저장까지 수행합니다. 앱 소스 배치는 다음 배치입니다.
  • 접속 확인
  • 서버 점검
  • 설치 방식
  • 사양 확정
  • 기본 구성
  • 결과
1단계

접속 확인 터미널에 나온 접속 확인 주소로 들어오면 이 브라우저만 설치 화면을 사용할 수 있습니다.

터미널에 출력된 접속 확인 주소로 접속하면 설치 권한 확인이 끝납니다.

접속 확인됨
서버 비밀번호 입력 없음

sudo g7inst setup으로 이미 설치 컨트롤러가 실행 중입니다.

확인
외부 포트 미개방

7717 포트는 SSH 터널로만 접근합니다. 방화벽에는 열지 않습니다.

확인
2단계

서버 상태 점검

신규 Ubuntu 서버만 안전하게 세팅할 수 있으므로 기존 웹서비스, 포트, 설치 흔적을 먼저 확인합니다.

아직 점검 전입니다. 점검 실행을 누르세요.

문제 해결 / 초기화 설치가 중간에 멈췄을 때만 사용합니다. 서버 전체 복구는 Lightsail 스냅샷이 기준입니다.

설치기 소유 흔적을 확인하는 중입니다.

    3단계

    설치 방식 선택

    일반 사용자는 권장 조합을 선택하고 다음으로 넘어가면 됩니다.

    설치 템플릿 자주 쓰는 서버 조합을 미리 적용합니다. 상세 설정에서 나중에 바꿀 수 있습니다.
    설치할 앱 서버 스택을 먼저 준비한 뒤 마지막 단계에서 올릴 앱 종류입니다. 현재 버전은 서버 패키지 준비와 리포트에 반영합니다.

    상세 설정
    4단계

    설치 사양 확정

    선택한 사양, 프로비저닝 계획, 변경 예정 항목을 확인한 뒤 기본 서버 구성 단계로 넘어갑니다.

    옵션을 확인한 뒤 계획 생성을 누르세요.
    5단계

    기본 서버 구성 실행

    apt 패키지, Nginx 도메인 연결, PHP-FPM/DB 튜닝, DB 계정, SSL, 설정 안내서를 순서대로 처리합니다.

    패키지 진행 상황

    설치 계획에서 확인한 apt 패키지를 하나씩 표시합니다.

    리포트 생성대기

    문제 해결 / 초기화 설치가 중간에 멈췄을 때만 사용합니다. 서버 전체 복구는 Lightsail 스냅샷이 기준입니다.

    설치기 소유 흔적을 확인하는 중입니다.

      6단계

      결과 리포트

      기본 서버 구성 결과, 검증 상태, 재시도 명령, 프로비저닝 계획을 확인합니다.

      리포트 새로고침서버에 저장된 결과만 다시 불러옵니다.
      설치 기록만 정리installer가 만든 설정/웹루트 기록을 기준으로 정리합니다. apt 패키지는 제거하지 않습니다.
