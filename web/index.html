현재 버전은 실제 패키지 설치 전 단계입니다. 설치 준비 파일, 계획, 리포트 생성까지 수행합니다.
  • 서버 로그인

1단계

서버 계정으로 로그인

설치 준비 작업은 root 또는 sudo 권한이 있는 서버 계정으로만 진행합니다.

서버 계정
2단계

서버 상태 점검

운영체제, 권한, 기존 웹서비스, 포트, 설치 흔적을 확인합니다.

아직 점검 전입니다. 점검 실행을 누르세요.
3단계

설치 방식 선택

일반 사용자는 권장 조합을 선택하고 다음으로 넘어가면 됩니다.

설치 템플릿 자주 쓰는 서버 조합을 미리 적용합니다. 상세 설정에서 나중에 바꿀 수 있습니다.
상세 설정
4단계

설치 사양 확정

선택한 사양과 변경 예정 항목을 확인한 뒤 설치 준비 단계로 넘어갑니다.

옵션을 확인한 뒤 계획 생성을 누르세요.

설치 준비 실행
